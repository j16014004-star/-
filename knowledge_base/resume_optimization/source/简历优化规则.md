# 简历优化规则

一份优秀简历应该做到：

1. 一页纸优先
2. 项目放在最重要位置
3. 每个项目包括：
- 项目简介
- 技术栈
- 负责内容
- 项目成果

描述时尽量使用：

设计
开发
实现
优化
部署
维护

避免：

负责...

推荐：

独立开发...
设计实现...

每一条最好体现成果。

例如：

Redis缓存热点数据，接口响应时间由600ms降低至120ms。

多使用数字：

提高30%
降低50%
支持1000并发
