# /remember - 恢复项目上下文

## 触发条件
当用户输入 `/remember` 时执行此技能。

## 执行步骤

### 1. 读取项目记忆文件
读取 `CLAUDE.md` 文件，获取完整的项目上下文信息，包括：
- 项目介绍与目标
- 技术栈与依赖版本
- 项目结构与文件说明
- 已完成模块清单
- 当前开发状态
- 已知 Bug 与待完成任务
- 编码规范与开发习惯
- 历史问题记录
- 重要决策记录

### 2. 输出上下文摘要
读取完成后，向用户输出简洁的上下文摘要，格式如下：

```
📋 项目上下文已恢复

🏗️ 项目: [项目名称]
📊 整体进度: [进度百分比]
✅ 已完成: [最近完成的功能]
🐛 当前 Bug: [已知 Bug 数量]
📝 待办事项: [待完成任务数量]
🚀 下一步: [建议的下一步操作]

准备好继续开发，请告诉我你要做什么。
```

### 3. 进入开发就绪状态
根据 CLAUDE.md 中的规则进入开发状态：
- 遵循编码规范
- 遵循用户的开发习惯偏好
- 遵循 Claude 开发协作规则
- 了解当前项目状态，可以立即开始工作

## 注意事项
- 如果 CLAUDE.md 文件不存在，提示用户创建
- 如果文件内容有误或过时，提醒用户更新
- 每次执行时都要完整读取文件，不要使用缓存
